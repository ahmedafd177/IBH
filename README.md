# InspiringBeautyHub

Nairobi's premier destination for authentic perfumes, luxury hair care & premium body products.

## Project Structure

```
IBH-project/
├── frontend/                # All frontend code
│   ├── index.html          # Main HTML file
│   ├── css/
│   │   └── style.css       # Stylesheets
│   ├── js/
│   │   └── script.js       # JavaScript functionality
│   └── assets/
│       └── images/         # Static assets (logos, images)
├── backend/                # Backend code (API, database, etc.)
│   └── README.md           # Backend setup guide
├── README.md               # This file
└── .gitignore              # Git ignore rules
```

## Features

- Product catalog with perfumes, hair care, and body products
- Shopping cart functionality
- Admin dashboard for managing products and orders
- Responsive design for mobile and desktop
- Search and filtering capabilities

## Development

This is a static website built with HTML, CSS, and JavaScript. No build process required - just open `index.html` in a browser.

## Deployment

The site can be deployed to any static hosting service like:
- GitHub Pages
- Netlify
- Vercel
- Firebase Hosting
